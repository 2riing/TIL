### 주피터 노트북 단축키 

`ctrl + enter` =>  셀 실행(**Run**) 

`shift + enter` => 셀 실행 후 아래칸으로 내려감 (**Run & enter**)

**초록색** : 편집 모드 (`방향키 : 라인`)

**파란색** : 셀렉트/선택 모드 (`방향키 : 셀`) (h: 단축키에 대한 설명) (b:아래에 셀추가) (a:위에 셀추가) (dd: 셀삭제)



계산을 다시 시작하고 싶을 때 ,  Kernel - Restart (단축키 : 00)

Restart & Clear output하면 싹 다 사라짐
